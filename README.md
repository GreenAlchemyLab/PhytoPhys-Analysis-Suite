# PhytoPhys-Analysis-Suite
Script R per l'analisi della fisiologia vegetale: elaborazione dati nutrizionali (LaquaTwin $NO_3^-$ e $K^+$), indici clorofilliani (SPAD) e cinetica della fluorescenza/efficienza fotosintetica (MPM-100). Include pipeline di pulizia dati e visualizzazione statistica avanzata.
